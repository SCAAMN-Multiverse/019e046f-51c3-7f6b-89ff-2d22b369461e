# Ciencia — Antifragilidad y convexidad (Taleb)

**Nassim Nicholas Taleb** define *antifragilidad* como la propiedad de sistemas que se fortalecen con el estrés — opuesto a frágil, distinto de robusto. *Convexidad* es la asimetría entre ganancias potenciales (limitadas o ilimitadas) y pérdidas potenciales (acotadas).

## Lección que el aparato extrae

Justifica el ciclo evolutivo con `[planes-posibles]` y `[ideas-nuevas]`: mantener alternativas vivas a coste limitado para captar upside grande cuando una resulta valiosa. Y justifica el modelo del piloto SCAAMN — coste mensual limitado, payback potencial de millones, asimetría convexa.
