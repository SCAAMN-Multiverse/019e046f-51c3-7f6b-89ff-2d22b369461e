# Sorolla — 50 ingenieros, elegido para arquitectura, Java 6→8 + microservicios

Intervención General del Estado, sistema **Sorolla** (gestión económica de portfolios estatales). 50 ingenieros dando mantenimiento y mejora al sistema. En 6 meses los **propios compañeros eligieron al humano** para pasarlo al departamento de arquitectura. Estando allí desbloqueó la migración **Java 6 → Java 8** y acompañó el paso a **microservicios**. Caso ilustrativo de cómo el reconocimiento técnico interno opera al margen del organigrama formal.
