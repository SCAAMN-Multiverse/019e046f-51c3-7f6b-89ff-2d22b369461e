# W2M — gestión de hoteles vía procesos batch (junio-agosto)

World2Meet (agencia turística) maneja un volumen muy grande: **miles de proveedores de hoteles, millones de hoteles, miles de millones de habitaciones**. El equipo necesitaba que ese volumen fuera exportable y manipulable en Excel por usuarios de negocio, para lo cual hubo que diseñar **procesos batch** (orquestación, reintentos, ventanas de ejecución, dependencias entre jobs, observabilidad, idempotencia). Con IA integrada en el flujo de trabajo (**ChatGPT + GitHub Copilot**), el producto se implementó **completo entre junio y agosto — dos meses** para un reto que sin IA habría sido mucho mayor en tiempo y esfuerzo. Ilustra la fase 2 (IA dentro del proyecto).
